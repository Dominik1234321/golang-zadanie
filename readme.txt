## zadanie-golang 

1. run command: go run main.go
2. write name of book
3. choose format of sorting works by revision (asc,desc)

result :

    autors sorted by name with their works sorted by revision

example :

    Irie, Taikichi:
        - Yamatoji junrei (4)
        - Butsuzō (4)
        - Yamato no nobotoke (2)
        - Koshoku Yamatoji = (2)
        - Manʼyō no hana o tazunete (2)
        - Man'yō no hana, Yamatoji (1)


    Ishimoto, Yasuhiro:
        - Hana (6)
        - Katsura (5)
        - Kokoku no jūichimen kannon (4)
        - Tokyo (2)
        - Ishimoto Yasuhiro ten (2)
